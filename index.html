東京駅の歴史

知ると少しだけ風格を感じるが、集合にはほぼ寄与しない

東京駅は日本の鉄道の中心の一つとして長く機能してきた駅で、赤レンガ駅舎の重みは「ここから旅が始まる」感を強める。しかし今回のミッションにおいて重要なのは、その重厚な歴史ではなく、3人が8:00に同じ場所へ着くことである。

つまり、駅史を読むことは感情には効くが、遅刻の防止には直結しない。それでも卒業旅行の出発点として、若干の荘厳さを演出する効果はある。

ホテル運営のこと

経営状況を気にしても、今夜のチェックインは速くならない

宿に泊まる側として本当に重要なのは、駅から近いこと、荷物を預けられること、そして11:00に静かに出発できることだ。運営会社の経営指標を読み込み始めると、旅行者は一瞬だけ投資家の顔になるが、部屋の広さもベッドの弾力もその場では変わらない。

したがって本ページの結論は単純である。IRを読むより先に、充電器の所在を確認した方が有益である。

京都駅周辺 or 京都タワーの論理式

自然言語の or は排他的とは限らない

2日目 11:20 の「京都駅周辺 or 京都タワーをぶらつく」は、論理式で書くと `K OR T` である。ここで `K` は京都駅周辺をぶらつく、`T` は京都タワーに寄る、を表す。

ただし日常日本語の `or` は排他的 `XOR` ではなく、ふつうは包摂的 `OR` として読める。したがって、眠気と残り時間が許せば `K = 1, T = 1` も成立する。つまり、駅周辺を歩いたあとにタワーへ行く世界線は論理的に合法である。

  • `K = 1, T = 0`: 駅ナカと土産で終える。
  • `K = 0, T = 1`: タワーへ一直線に行く。
  • `K = 1, T = 1`: 駅周辺もタワーも回収する。
  • `K = 0, T = 0`: 眠すぎるのでベンチで虚無になる。

フーリエ級数展開

新幹線には効かないが、情報系の心には効く

周期 `2π` の関数 `f(x)` は、おおまかに言えば `a0/2 + Σ(an cos nx + bn sin nx)` の形に分解できる。波を細かい正弦波と余弦波の和で見る発想であり、旅程の複雑さを周波数成分に分けることはできないが、気分だけは理系になる。

たとえば矩形波なら奇数次の正弦項だけが残り、打ち切り近似でもそれっぽい形が出る。なお、この知識を持っていても 9:00 発ののぞみに乗り遅れることはある。

function squareWaveApprox(x, terms = 10) {
  let sum = 0;
  for (let k = 1; k <= terms; k += 1) {
    const n = 2 * k - 1;
    sum += Math.sin(n * x) / n;
  }
  return (4 / Math.PI) * sum;
}
